# Fitness Training Data Analysis

## Описание проекта

Проект анализирует эффективность различных тренировочных программ на основе фитнес-данных.

В проекте сравниваются:
- синтез белка
- прирост силы
- гормональный фон
- тренировочный объём
- интенсивность тренировок

Проект позволяет получить аналитические выводы через CLI-интерфейс.

# Возможности проекта

## Аналитика

Проект реализует следующие выводы:

1. Эффективность программ тренировок по синтезу белка
2. Эффективность программ тренировок по росту силы
3. Эффективность программ тренировок по гормональному фону
4. Универсальная программа тренировок
5. Корреляция объёма тренировок и синтеза белка
6. Сравнение высокоинтенсивных и низкоинтенсивных тренировок

# Запуск проекта

## Установка зависимостей

```bash
pip install -r requirements.txt
```

---

# Запуск CLI

## Эффективность по синтезу белка

```bash
python main.py protein
```

## Эффективность по силе

```bash
python main.py strength
```

## Гормональная эффективность

```bash
python main.py hormones
```

## Универсальная программа

```bash
python main.py universal
```

## Корреляция объёма и синтеза белка

```bash
python main.py corr
```

## Сравнение интенсивности

```bash
python main.py intensity
```
